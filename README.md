# YouTube Downloader Telegram Bot

هذا بوت تليجرام متقدم لتحميل الفيديوهات من يوتيوب بجودات مختلفة، مع دعم قص الفيديو وتحويله إلى MP3.

## المميزات
- تحميل الفيديوهات بجودات (360p, 480p, 720p, 1080p).
- تحويل الفيديوهات إلى ملفات صوتية MP3 (128kbps, 320kbps).
- إمكانية قص جزء محدد من الفيديو عن طريق إرسال وقت البداية والنهاية.
- نظام قائمة بيضاء (Whitelist) للسماح لمستخدمين محددين فقط.
- نظام حد الطلبات (Rate Limiting) لمنع الإساءة (3 طلبات في الساعة).
- جاهز للتشغيل على Render باستخدام Docker.

## كيفية التشغيل
1. قم بتثبيت التبعيات: `npm install`
2. تأكد من تثبيت `ffmpeg` و `yt-dlp` على جهازك.
3. قم بتعديل ملف `.env` وإضافة التوكن الخاص بك.
4. ابدأ البوت: `npm start`

## التشغيل باستخدام Docker
```bash
docker build -t yt-downloader-bot .
docker run -d --env-file .env yt-downloader-bot
```
